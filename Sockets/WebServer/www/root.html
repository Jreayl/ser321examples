You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
      • Example: jrlaytonser321.duckdns.org/file/sample.html

  • /json -- returns a json of the /random request
      • Example: jrlaytonser321.duckdns.org/json

  • /random -- returns index.html
      • Example: jrlaytonser321.duckdns.org/random

  • /multiply -- returns product of two integers
      • Example: jrlaytonser321.duckdns.org/multiply?num1=10&num2=20

  • /github -- pulls information from github
      • Example: jrlaytonser321.duckdns.org/github?query=users/amehlhase316/repos

  • /video -- plays a cool video and lets you specify frame size
      • Example: jrlaytonser321.duckdns.org/video?width=900&height=600

  • /button -- creates a button and lets you specify color and text
      • Example: jrlaytonser321.duckdns.org/button?color=purple&text=Click Me!
